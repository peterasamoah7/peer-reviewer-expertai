Peer Reviewers App

Uploading a file

The Peer Revievers App helps academic scholars and instituitions to select Peer Reviewers for a submitted paper using document similarity. The content of the upload file is extracted and analyzed using the Expert AI KeyPhrase Extraction API.

Extracting keyphrases and topics....

Building an intelligent query

The Topics represented the extracted topics from the KeyPhrase API and represent what the content is about. The Phrases represent the key phrases from the submitted documented. Click the high scoring topics or phrases to build an intelligent query for searching the document index.

Searching for matching documents....

Search Results

The search results represent the best matching documents based on the query that was built. The authors of the papers with the highest scoring document similarity scores are the best possible reviewers for the research paper submitted.
